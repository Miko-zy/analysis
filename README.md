# 智能数据分析系统 🚀mcp接入小智/部分成功调用

基于本地大模型的数据分析平台，支持自然语言查询、交互式可视化和AI深度分析。

## 功能特性 ✨

### 📊 数据管理
- 支持 MySQL 数据库连接
-数据表浏览与预览
- 表结构查看
- 数据摘要统计

### 🔍 智能查询
- 自然语言转SQL查询
-SQL查询验证与解释
-查询结果实时展示
-查询历史记录

### 📈 数据分析
- 多种可视化图表（柱状图、折线图、散点图等）
- 交互式数据探索
- 异常值检测
- 回归分析
- 时间序列分析

### 🤖 AI 分析
- 智能数据洞察
- 业务建议生成
- 趋势预测
- 数据质量评估

### ⚙️ 系统管理
- 数据库连接测试
- LLM 连接测试
- 日志管理
- 系统状态监控
